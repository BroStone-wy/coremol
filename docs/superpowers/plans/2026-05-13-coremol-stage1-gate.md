# CoReMol Stage-1 Gate Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build and run the BBBP/ESOL three-seed CoReMol gate with task metrics, TCM@K, and failure analysis.

**Architecture:** `CORMOL` is an isolated PyTorch Geometric project. Dataset loading, scaffold splitting, model modules, training, probing, and reporting are separate files with explicit inputs and outputs. CoReMol modifies communication inside the model instead of rewiring the molecular graph.

**Tech Stack:** Python 3.9, PyTorch, PyTorch Geometric, RDKit, scikit-learn, pandas, PyYAML, pytest.
